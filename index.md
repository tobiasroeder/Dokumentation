## Installation

Add the Dokumentation CSS file.

### CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/dokumentation@0.1.0/dokumentation.min.css" />
```

Or install it with NPM.

### NPM

```shell
npm install dokumentation
```
